Git is a very versatile tool which uses bash to access git repositories. These repositories are hosted on GitHub. Through git you can clone a repository, You can push changes made locally to the main project. You can create files and send it to the GitHub repository. Git can be accessed through bash commands. Git can be accessed through the command in line or a GUI. It supported by multiple  platforms, windows, mac and Linux etc. This powerful tool allows developers to maintain their projects. This helps teams organize and maintain source code or any project structure of files or folders.
